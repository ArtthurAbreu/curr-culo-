Projetos

BRCLIGHTNING - Integração 3D para dimensionamento de SPDA

O projeto teve como objetivo principal desenvolver uma metodologia para integrar dados de dimensionamento de SPDA em formato 3D, superando as limitações da abordagem bidimensional. O sistema permite visualizar o volume de proteção do SPDA em 3D, além de gerar imagens de estruturas de captação complexas, utilizando three.js biblioteca do Java Script para criação dos modelos 3D. Papel fundamental no projeto, foi o desenvolvimento da interação do usuário ao sistema, utilizando HTML, CSS e Java Script.

AgendaService

O projeto propõe um sistema inovador para agendamento e monitoramento de serviços mecânicos automotivos, focado na interface e na usabilidade. Imagine uma plataforma como um GPS moderno: intuitiva, transparente e fácil de usar. Ela permitirá aos clientes agendar serviços online e acompanhar a fila de atendimento em tempo real com uma interface intuitiva. Esse sistema não só reduzirá os períodos de espera, mas também melhora a gestão do fluxo de trabalho da empresa, proporcionando uma experiência superior ao cliente e uma operação mais eficiente. Tecnologia Next.js foi utilizada na criação das interface do sistema.
